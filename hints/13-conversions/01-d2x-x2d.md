`D2X(number)` converts a decimal (base-10) number to hexadecimal (base-16):

```rexx
SAY D2X(255)   /* prints: FF */
SAY D2X(10)    /* prints: A */
SAY D2X(0)     /* prints: 0 */
```

`X2D(hexstring)` converts a hexadecimal string back to decimal:

```rexx
SAY X2D('FF')  /* prints: 255 */
SAY X2D('A')   /* prints: 10 */
```

Note that the hex string argument to `X2D` should be in quotes since
it is a string, not a number.
