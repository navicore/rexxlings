`B2X(binarystring)` converts a string of binary digits to hexadecimal:

```rexx
SAY B2X('1111')       /* prints: F */
SAY B2X('11111111')   /* prints: FF */
SAY B2X('10100101')   /* prints: A5 */
```

The binary string length should be a multiple of 4 (one hex digit per
4 binary digits). Leading zeros can be omitted.

`X2B(hexstring)` converts a hex string to binary (4 bits per hex digit):

```rexx
SAY X2B('F')    /* prints: 1111 */
SAY X2B('FF')   /* prints: 11111111 */
SAY X2B('1')    /* prints: 0001 */
```
