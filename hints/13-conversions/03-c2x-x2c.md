# Hint: C2X() and X2C() â€” Character/Hex Conversion

`C2X(string)` converts each byte of a character string to its 2-digit
hexadecimal representation:

```rexx
SAY C2X('A')     /* prints: 41 (hex for ASCII 65) */
SAY C2X('Hi')    /* prints: 4869 */
```

`X2C(hexstring)` converts pairs of hex digits back to characters:

```rexx
SAY X2C('41')    /* prints: A */
SAY X2C('4869')  /* prints: Hi */
```

Useful hex codes to know:
- `41`-`5A` = A-Z
- `61`-`7A` = a-z
- `30`-`39` = 0-9
- `4F` = O, `4B` = K
