# Hint: MAX() and MIN() with Multiple Arguments

`MAX()` and `MIN()` accept two or more numeric arguments and return the
largest or smallest value, respectively:

```rexx
SAY MAX(3, 7, 2)      /* prints: 7 */
SAY MIN(3, 7, 2)      /* prints: 2 */
SAY MAX(-5, -1, -10)  /* prints: -1 */
```

Replace each `???` with the appropriate function call, using the numbers
that will produce the expected result. Look at the test assertions to
see which arguments to use.

For example:

```rexx
biggest = MAX(10, 42, 7, 33)
```
